# Example Data

Three SEM micrographs of porous material cross-sections acquired at 1000x magnification.

- **sample_1.tif** — Porous microstructure with distributed pore network
- **sample_2.tif** — Cross-section showing varied pore morphology
- **sample_3.tif** — Cross-section with distinct pore size distribution

**Image specifications:**
- Format: TIFF (grayscale)
- Dimensions: 1536 x 1103 pixels
- Magnification: 1000x
- Horizontal field width: 207 µm
- Calibration: 7.42 pixels/µm
